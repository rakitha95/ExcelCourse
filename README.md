אודות הקורס

תרגילים מעשיים ודפי עבודה
לכל מודול מצורף דף עבודה. כל דף עבודה כולל:
- הסבר קצר בעברית.  
- מקורות נתונים.  
- משימה ברורה.  
- פיתרון מובנה להצלבה.  

דוגמה לתרגיל:
- שם: דוח מכירות חודשי  
- נתונים: טבלת מכירות עם תאריכים, מוצר, אזור, כמות, מחיר יחידה  
- משימה: חשבו הכנסות לפי אזור ולפי חודש. צרו PivotTable שמציג סכום הכמות וסכום ההכנסה.  
- מטרת התרגיל: להכיר PivotTable, קבוצת תאריכים, ויחידות חישוב.  

פניות פורמליות: פורמט קבצים והפעלת מאקרו
פריטים חשובים:
- שמרו גרסה נקייה לפני שמירה כ-.xlsm.  
- שמרו גיבוי של קבצי המקור.  
- בעת הפעלת מאקרו, בדקו את שם הפונקציה הראשית בגליון.  
- אם אתם עובדים ב-Mac, חלק מהמאקרו עשוי לדרוש התאמות.  

נוסחאות מרכזיות עם דוגמאות
להלן רשימה מפורטת של נוסחאות שכדאי לשלוט בהן. לכל נוסחה יש דוגמה קצרה.

SUM
- שימוש: סוכם טווח מספרים.  
- דוגמה: =SUM(B2:B20) — סכום רכיבי הטווח.

AVERAGE
- שימוש: ממוצע מספרים.  
- דוגמה: =AVERAGE(C2:C20)

COUNT, COUNTA
- COUNT סופר ערכים מספריים.  
- COUNTA סופר ערכים לא ריקים.  
- דוגמה: =COUNT(A2:A100), =COUNTA(A2:A100)

IF
- תנאי פשוט.  
- דוגמה: =IF(B2>1000, "גבוה", "נמוך")

SUMIFS / COUNTIFS
- סכום לפי מספר תנאים.  
- דוגמה: =SUMIFS(D:D, A:A, "מוצר A", B:B, ">=2024-01-01")

VLOOKUP
- חיפוש אנכי בטבלה.  
- דוגמה: =VLOOKUP(E2, Products!A:D, 3, FALSE)

XLOOKUP
- חיפוש מודרני עם יתרונות על VLOOKUP.  
- דוגמה: =XLOOKUP(E2, Products!A:A, Products!C:C, "לא נמצא")

INDEX + MATCH
- חלופה גמישה ל-VLOOKUP.  
- דוגמה: =INDEX(Products!C:C, MATCH(E2, Products!A:A, 0))

FILTER, UNIQUE, SORT
- כלים לעיבוד דינמי בטבלאות חדשות.  
- דוגמה: =FILTER(Data!A:C, Data!B:B="Active")

LET, LAMBDA
- LET שומר תוצאות בשם בתוך נוסחה.  
- דוגמה LET: =LET(x, SUM(A2:A10), x/COUNT(A2:A10))

דוגמאות נוספות
- TEXT — המרת תאריך למחרוזת.  
- DATE, YEAR, MONTH, DAY — עבודה עם תאריכים.  
- EOMONTH — סוף חודש.  
- CONCAT / TEXTJOIN — איחוד טקסטים.

טבלאות פיווט — מדריך שלב אחר שלב
טבלת PivotTable תאפשר לכם לסכם נתונים משתנים. הנה שלבים ברורים.

1. הכנת מקורות הנתונים
- ודאו שכל עמודה כוללת כותרת.  
- הוציאו שורות ריקות.  
- צרו טבלה (Ctrl+T) כדי לשמור על טווח דינמי.

2. יצירת PivotTable
- בחרו תא בתוך הטבלה.  
- Insert → PivotTable.  
- בחרו מקום חדש או גליון קיים.

3. שדות וארגון
- גררו שדות ל-Rows ו-Columns לקבלת חיתוך.  
- גררו שדות ל-Values כדי לסכם.  
- השתמשו ב-Slicers לפילטור חזותי.

4. חישובי שדה
- הוספת שדה מחושב: PivotTable Analyze → Fields, Items & Sets → Calculated Field.  
- דוגמה: רווח ממוצע = הסכום של הכנסות לחלק לכמות.

5. קבוצת תאריכים
- קליק ימני על תאריך → Group.  
- בחרו חודשים, רבעונים, שנים.

6. עיצוב והצגה
- השתמשו ב-Design → Report Layout → Show in Tabular Form.  
- הוסיפו סיכומים ופורמטים מותאמים.

7. דינמיות
- עדכון PivotTable לאחר שינוי נתונים: Refresh.  
- השתמשו ב-PivotCache עבור ביצועים טובים יותר עם טבלאות גדולות.

Power Query וייבוא נתונים
Power Query יעיל לניקוי וטרנספורמציות. עקרונות עבודה:

1. שאילתה חדשה
- Data → Get Data → From File / From Web / From Database.

2. פעולות נפוצות
- Remove Columns, Rename Columns.  
- Replace Values, Trim, Split Column.  
- Unpivot Columns כדי להפוך טבלה רחבה לצורת עמודה.

3. שרשור ואיחוד
- Append = חיבור טבלאות בעלות מבנה זהה.  
- Merge = חיבור על בסיס טור משותף כמו SQL Join.

4. טעינה ל-Excel
- Load To → Table / PivotTable Report / Only Create Connection.

דשבורדים וויזואליזציה
עקרונות עיצוב:
- מטרה ברורה לכל דשבורד.

מאקרו ו-VBA — מבוא ותרגול

2. עריכה
- Alt+F11 פותח את העורך.

3. דוגמאות שימוש
- יצירת דוח PDF אוטומטי.  
- עדכון נתונים מכל טווח חיצוני.

שאלות נפוצות (FAQ)
Q: באיזה גירסאות של Excel הקורס תואם?  
A: הקורס תואם ל-Excel 2016 ומעלה. חלק מהנוסחאות הדינמיות דורשות Excel 365 או גרסה שמכילה את הפונקציות החדשות.

Q: האם הקבצים מכילים מאקרו?  
A: כן. חלק מהקבצים כוללים מאקרו (.xlsm). הוראה לגבי הפעלת מאקרו מופיעה בתוך הקובץ ובריליס.

Q: איך ניתן לקבל קבצים באנגלית במקום בעברית?  
A: ניתן ליצור ענף (branch) או לבקש pull request עם קבצים מתורגמים. הוראות תרומה למטה.

Q: האם יש מבחנים או תעודה?  
A: הקורס כולל תרגילים ופרויקטים. תעודה לא מסופקת באופן אוטומטי. ניתן לקבל משוב על פרויקטים דרך פתיחת Issue.

איך לתרום
- פתחו Issue אם מצאתם באג או שגיאת תוכן.  
- שלחו Pull Request עם שיפור תוכן, תרגום או תוספת תרגיל.  
- השתמשו ב-branch על מנת לשמור על היסטוריה נקייה.  
- טאגו PR עם כינוי הקורס והמודול המותאם.
