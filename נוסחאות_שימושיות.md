## 住转 住住转

### 砖 转
```
=A1+B1          # 专
=A1-B1          # 住专
=A1*B1          # 驻
=A1/B1          # 拽
=A1^2           # 拽
=A1*1.1         # 住驻转 10%
=A1*0.9         # 驻转转 10%
```

### 驻拽爪转 住住转
```
=SUM(A1:A10)    # 住
=AVERAGE(A1:A10) # 爪注

=MAX(A1:A10)    # 拽住
=MIN(A1:A10)    #

```

## 驻拽爪转 拽住

### 专 拽住
```

=A1&" "&B1               # 专 注 &

=MID(A1,2,4)             # 4 转 拽 2
=LEN(A1)                 # 专 拽住

```

### 驻转 拽住

```
=REPLACE(A1,1,3,"砖")      # 驻转 转
```
